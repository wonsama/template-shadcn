# README

`nextjs14 + shadcn/ui + tailwind` 조합의 프로젝트를 손쉽게 시작하기 위한 template

## 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 결과를 확인하세요.
